# Sales Data Analysis Project

## Project Overview
This project analyzes sales data to uncover trends, identify top-selling products, and evaluate regional performance. The goal is to provide actionable insights for business decision-making.

## Tools & Technologies
- **Python** for data analysis
- **Pandas** for data manipulation
- **Matplotlib & Seaborn** for data visualization

## Dataset Description
The dataset contains the following columns:
- **Order Date** – Date of the sale
- **Product** – Name of the product sold
- **Quantity** – Number of units sold
- **Price** – Price per unit
- **Region** – Region where the sale occurred

## Key Insights
- **Highest Sales Month:** March recorded the maximum total sales.
- **Top Product:** Laptop was the best-selling product.
- **Best Region:** North region contributed the most revenue.
- **Sales Trend:** Overall sales showed an increasing trend from January to April.

## Visualizations
- **Bar Chart:** Monthly Sales Trend  
- **Bar Chart:** Sales by Product  
- **Pie Chart:** Sales by Region  

## How to Run
1. Place `sales_data.csv` in the `data/` folder.  
2. Open `sales_analysis.ipynb` in Jupyter Notebook.  
3. Run all cells to see analysis and visualizations.
